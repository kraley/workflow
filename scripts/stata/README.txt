This README file provides instructions on how to run the code in this repository.

You will need to edit the setup_example.do file to set global macros to directories on 
your commputer. Save the script as setup_<yourusername>. <yourusername> is your username on this computer.

Change to the workflow/scripts directory before starting stata or within stata.

do marcoh_trends.do 

After the scripts run, you should find the results in the directory assigned in the 
results global macro. The log should be in the directory assigned in the logdir 
global macro (set in setup_<yourusername>
